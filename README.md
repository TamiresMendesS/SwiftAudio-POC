## SwiftAudio-POC

Esse repositório foi desenvolvido exclusivamente para a exploração do framework AVFoundation, bem como a linguagem Swift, desenvolvendo uma POC com elementos de SwiftUI, com objetivo de aprender a utilizar o framework e aplicá-lo com eficiência.

## Ferramentas Utilizadas

- XCode
- Swift
  - SwiftUI
  - AVFoundation
